# MixMasterOS Worker

Internal background worker for MixMasterOS. Polls Supabase for queued audio jobs, runs the private DSP engine, and uploads results back to Supabase Storage.

## Deploy on Render (Background Worker)

1. Push this folder to a GitHub repo.
2. On Render: New → Background Worker → connect repo.
3. Runtime: Docker (uses included `Dockerfile`).
4. Environment variables:
   - `SUPABASE_URL`
   - `SUPABASE_SERVICE_ROLE_KEY`
5. Deploy.

The container runs `python worker/main.py` and polls every 3 seconds.
